# 디자인 시스템

## 코드 블록
- 화려한 그라데이션 배경 지양
- 기본 테마 스타일 선호
- 가독성 우선

## 레이아웃
- **본문 너비**: 적절한 확장 (900-950px)
- **좌우 여백**: 과하지 않게 (1.5rem)
- **글꼴 크기**: 16px (가독성과 정보량 균형)

## 모던 요소
- 제목 그라데이션 ✅
- 호버 애니메이션 ✅
- 인라인 코드 스타일 ✅
- 링크 효과 ✅
- 구분선 (---) 활용 ✅

## 디자인 원칙
- **과한 디자인 요소 시 즉시 수정 요청 예상**
- **CSS 수정 시 단계별 적용 및 확인**
- **가독성 우선**: 디자인보다 사용성 중심
